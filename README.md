# ResolveX - Ticket Management System

ResolveX is a simple ticket management system built using FastAPI and a basic HTML/CSS/JS frontend.

## Features
- Customer & Agent Login/Signup
- Session-based login using localStorage
- Create Ticket popup modal
- Auto-generated Ticket IDs
- Ticket Dashboard view

## Tech Stack
Backend: FastAPI  
Frontend: HTML, CSS, JavaScript  
Database: JSON files

## How to Run

### Backend
```bash
uvicorn main:app --reload
