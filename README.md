# ResolveX Ticket Management System

Welcome to the ResolveX Ticket Management System project! This application is designed to help users manage and track their tickets efficiently. The system allows users to create, view, update, and delete tickets, providing a streamlined process for both users and administrators.

## Features
- **User Authentication**: Secure login/signup for users.
- **Ticket Creation**: Users can create new tickets with relevant details.
- **Ticket Management**: Users can view, update, or delete existing tickets.
- **Admin Panel**: Administrators can view all tickets and manage user accounts.
- **Search Functionality**: Quickly find tickets based on various criteria.
- **Status Tracking**: Tickets can have different statuses (open, in progress, closed).

## Technology Stack
- **Backend**: FastAPI
- **Database**: SQLite (or other databases can be used.)
- **Frontend**: HTML/CSS and JavaScript (if applicable) 

## Getting Started
To get started with the ResolveX Ticket Management System, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Sandip7224/ResolveX_FastAPI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ResolveX_FastAPI
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   uvicorn app.main:app --reload
   ```
